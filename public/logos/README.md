# MP Consulting Logo Options

This folder contains 5 alternative logo designs for MP Consulting Limited.

## Logo Files

- **logo-option-1.svg** - Classic Box Design
- **logo-option-2.svg** - Circular Badge Design
- **logo-option-3.svg** - Accent Line Design
- **logo-option-4.svg** - Bordered Badge Design
- **logo-option-5.svg** - Geometric Shape Design

## Preview

Open `logo-preview.html` in your browser to see all logo options side-by-side with descriptions.

## How to Use

1. Choose your preferred logo option
2. The SVG files can be used directly in your website
3. To implement in the Header component, replace the current text-based logo with an `<Image>` or `<img>` tag pointing to the selected logo

## Color Scheme

All logos use the brand colors:
- **Primary Navy**: #243b53 / #102a43
- **Accent Gold**: #ffc91a

## File Format

All logos are in SVG format, which means:
- Scalable to any size without quality loss
- Small file sizes
- Perfect for web and print
- Can be styled with CSS if needed
